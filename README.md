> ⚠️ **Important:**

## ⚙️ How USDT Flashing Works

1. **Wallet Address Input**  
   The user enters a wallet address (e.g., Trust Wallet, MetaMask).

2. **Fake Transaction Creation**  
   The app simulates a transfer but does NOT broadcast it to the blockchain.

3. **Temporary Balance Display**  
   Some wallets may show a fake balance due to:
   - Custom tokens  
   - Manipulated RPC nodes  
   - UI spoofing  

4. **No Blockchain Confirmation**  
   The transaction cannot be verified on blockchain explorers.

5. **Funds Are Not Real**  
   The displayed USDT:
   - Cannot be sent  
   - Cannot be swapped  
   - Eventually disappears
